# 409kt

This is a project aim to create a comprehensive tool helping to organize collections, create aids,
help with listbuilding of various Games Workshop game systems. The application comes without any
data of units or rules as they are part of GW intelectual property and you should buy proper
book to access them.

The application is written using React.js and recoil.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `npm test`
